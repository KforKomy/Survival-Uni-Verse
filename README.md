# 🎮 2D Game Development Project

## 📌 Overview

This project is a 2D game developed in **C++ using SFML**, focusing on implementing core gameplay mechanics while maintaining a clean, modular code structure. The goal was to build a responsive and efficient game while applying software engineering best practices.

---

## 🚀 Features

* 🎯 Player movement system with smooth controls
* ❤️ Health system for player state management
* 🤖 Basic enemy AI behavior
* 🏆 Score tracking system
* 💥 Collision detection using SFML
* 🎮 Real-time input handling
* 💾 Saving and loading game data using C++ file handling

---

## 🛠️ Technologies Used

* **C++**
* **SFML (Simple and Fast Multimedia Library)**

---

## 🧩 Implementation Details

* Designed and implemented core game mechanics from scratch
* Used SFML for rendering, input handling, and collision detection
* Structured the project using a modular architecture to improve maintainability
* Applied performance optimizations to ensure smooth gameplay

---

## ▶️ How to Run

1. Install **SFML** on your system
2. Clone the repository:

   ```bash
   git clone https://github.com/KforKomy/Survival-Uni-Verse.git
   ```
3. Compile the project using your preferred compiler
4. Run the executable
